Токен читается через бат файл(из виртуального окружения).
Метод os.getenv("имя_переменной") используется для получения значения переменной окружения с заданным именем. 
Переменные окружения - это некие "переменные", хранящие информацию, доступную программам, выполняющимся в операционной системе.

В бат-файле токен мы задаем без кавычек:
	set TOKEN=6511163742:AAFRWyTNRyJ2nuHNWvNT8pzXWpWG3-4mnAo
в файле бота импортируем модуль os
	import os
Для начала инициализируем нашего бота
в него читаем наш токен.Пишем token= далее нам понадобится os и метод .getenv и в него записываем имя 
переменной(ее назвали TOKEN),ОБЯЗАТЕЛЬНО В КАВЫЧКАХ,Т.К ТОКЕН В БАТ-ФАЙЛЕ ЗАПИСАН БЕЗ
	bot = Bot(token=os.getenv("TOKEN"))

	TOKEN = '6511234164:AAEgrC6a2atS2yzTCUPr5HcLjZsyfP1btpk'
